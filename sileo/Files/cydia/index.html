Cydia Installer

 

Hosted on repo.cf/sileo, this version of the Cydia Installer was originally hosted on Procursus, before being taken down.

You will only view this depiction on Sileo/Sileo-based package managers.

You will probably encounter issues such as APT being unable to find the .deb file for this Cydia. Try installing Zebra first and use that to install to bypass this issue.
